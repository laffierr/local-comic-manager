# ComicLibrary 漫画图书馆

Android 漫画管理应用，帮助你整理、浏览和阅读本地漫画收藏。

## 功能列表

### 漫画管理
- **导入漫画** — 选择包含图片的文件夹添加为漫画，自动扫描封面和页数
- **浏览书架** — 网格/列表展示所有漫画，支持封面预览
- **搜索漫画** — 按标题搜索已导入的漫画
- **标签筛选** — 点击搜索栏标签图标展开面板，支持多标签 AND 逻辑筛选，筛选结果显示为列表
- **删除漫画** — 单个删除或批量多选删除（删除后显示 Snackbar 提示）
- **收藏切换** — 长按漫画封面可标记/取消收藏

### 标签系统
- **标签管理** — 创建、重命名、删除标签
- **漫画打标签** — 在漫画详情页为漫画添加/移除标签
- **批量打标签** — 多选漫画后批量添加标签或新建标签
- **标签视图** — 书架页按标签分组展示漫画行，点击"全部"查看该标签下所有漫画

### 收藏夹
- **创建收藏夹** — 自定义名称和描述
- **编辑收藏夹** — 修改名称和描述
- **置顶收藏夹** — 重要收藏夹固定在列表顶部
- **收藏夹内容** — 查看收藏夹内漫画，支持排序和分页浏览
- **批量操作** — 多选收藏夹进行批量删除
- **我的收藏** — 自动创建的"我的收藏"收藏夹（收藏的漫画自动加入）

### 阅读器
- **两种阅读模式** — 滚动模式（上下滑动）和翻页模式（左右翻页）
- **页面跳转** — 底部滑动条快速跳转到指定页码
- **全屏阅读** — 沉浸式阅读体验
- **封面入口** — 漫画详情页点击封面即可开始阅读

### 批量导入
- **一键导入** — 选择一个父文件夹，其中所有子文件夹自动导入为漫画
- **自动创建收藏夹** — 以父文件夹名称创建收藏夹，所有导入的漫画自动归入
- **去重检测** — 跳过已导入的漫画文件夹
- **进度显示** — 导入过程中显示实时进度

### 主题设置
- **外观模式** — 跟随系统 / 深色 / 浅色三种模式
- **主题颜色** — 紫色、蓝色、绿色、玫红、橙色、青色六种配色
- **全局生效** — 主题更改后整个应用即时切换

### 其他功能
- **网格密度** — 收藏夹详情和标签漫画页的漫画网格支持 3/4/5 列切换
- **首页推荐** — 随机推荐漫画和最近添加的漫画
- **划动切换** — 书架和收藏夹之间支持左右划动切换
- **数据持久化** — Room 数据库 + DataStore 偏好设置，数据安全可靠

## 技术栈

| 组件 | 技术 |
|------|------|
| 语言 | Kotlin |
| UI | Jetpack Compose (Material 3) |
| 架构 | MVVM + Repository |
| 依赖注入 | Hilt |
| 数据库 | Room |
| 偏好存储 | DataStore Preferences |
| 图片加载 | Coil 3 |
| 导航 | Navigation Compose + HorizontalPager |
| 文件访问 | Storage Access Framework (SAF) |
